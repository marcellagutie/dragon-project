## Como rodar o projeto?

Vem comigo que vou te ensinar.

### Usando npm

  1. Clone o repositório
  2. Rode o comando `npm install` 
  3. Por fim, rode `npm start`
  4. Agora é só acessar `http://localhost:3000/` para ver as previsões do tempo
